# Fluid Bootstrap 4 Theme

## Getting started

2. Clone this repo
3. Run `npm install`
4. Run `gulp serve`

A browser tab will be opened with a live preview of the theme. Changing files will trigger a rebuild and the live preview will be refreshed.
